# Sales & Revenue Dashboard (Power BI)

This project is an interactive dashboard built using Power BI to analyze sales performance across products, regions, and categories.

## Features:
- Total Sales KPI
- Region-wise analysis
- Product-wise sales
- Category insights
- Interactive slicer filters

## Tools Used:
Power BI, Power Query, Excel

## File:
Sales_Dashboard.pbix

The dataset used in this project contains sales transaction data with the following fields:

Order ID
Product
Category
Region
Quantity
Unit Price
Date

The dataset was used to analyze sales performance and build interactive visualizations in Power BI.
